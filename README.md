# zh-cli

> 前端开发脚手架

## 相关文档

- [nodejs 工具开发(一)](https://zhuanlan.zhihu.com/p/40223877)

## 快速开始

```bash
npm install -g zh-cli // 安装
npm create <project-name> // 创建项目
```
