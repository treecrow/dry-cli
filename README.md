## cli 功能

| command                                    | more                       |
| ------------------------------------------ | -------------------------- |
| create <projectName>                       | 项目初始化 项目名          |
| Check the features needed for your project | pwa TypeScript UnitTesting |
| path 模式                                  | history / hash             |
| output.publicPath                          | 项目静态资源放置的目录     |
